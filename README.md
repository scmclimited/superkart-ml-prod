# superkart-ml-prod Description
Dockerized Juypyter Notebook - ML Inference with REST APIs and Frontend


## Usage commands - Kickstart local services context
### Start all services
docker-compose up -d

### View logs
docker-compose logs -f [service-name]

### Stop all services
docker-compose down

### Rebuild and restart
docker-compose up -d --build

### Check service health
docker-compose ps
